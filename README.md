# Overview

Building on top of the concept laid out by Fei Protocol, Essence Finance aims to realize a vision of issuing a stablecoin that is both highly scalable and truly decentralized.

Essence Finance is unlocking a new paradigm for decentralized finance, one that is focused on providing users with a stable and decentralized way to store and transfer value, without sacrificing the core principles of decentralization and security.

## Tokens
The Essence Ecosystem encompasses a dual token model. Made up of:
-   **CHI:** A fully collateralized, yet scalable, decentralized stablecoin pegged to the US dollar.
-   **ZEN:** A dual utility token serving as the governance token and beneficiary of protocol growth.
